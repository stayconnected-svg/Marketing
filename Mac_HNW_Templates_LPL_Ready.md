### Banned Terms Avoided


| Banned Term                           | Replacement Used             |
| ------------------------------------- | ---------------------------- |
| guaranteed / secure / safe            | "pursue," "seek," "may help" |
| achieve / ensure / meet (as promises) | "work toward," "seek to"     |
| trusted / objective / unbiased        | "independent," "experienced" |
| unique / innovative / revolutionary   | Not used                     |
| peace of mind / sleep at night        | Not used                     |
| maximize / optimize (as certainty)    | "stronger growth potential"  |
| eliminate risk                        | "manage," "reduce exposure"  |
| best / superior / top / elite         | Not used                     |
| expert / guru                         | "specialist"                 |
| don't miss out / act fast             | Not used                     |
| proven / solid                        | Not used                     |
| protect / protected (as promise)      | "seek to manage"             |
| financial security                    | "financial future"           |
| solution / solve (tied to product)    | "strategies," "approach"     |
