# Image Legality Analysis — Intelligent Annuity Solutions Redesign

## Platform Context

Bryan McInerney's current site (checkannuity.com) is built on **FMG Suite**, a financial advisor website/marketing platform. FMG Suite provides:
- Pre-built templates and page layouts
- A licensed stock photo library (for use ONLY within FMG Suite sites)
- Compliance-reviewed content modules
- Resource center content (articles, videos, calculators)

This means: **Most images on the current site are FMG Suite licensed assets that CANNOT be reused outside their platform.**

## Recommendations for Production Build

### Immediate (Demo Phase — Current State)
- ✅ All images in demo use **Unsplash** (free, commercially licensed under Unsplash License)
- ✅ SVG owl logo created as interpretation (not a copy of original)
- ✅ Headshot shown as placeholder awaiting Bryan's file
- ✅ YouTube embeds are Bryan's own videos — fully legal to embed

### For Final Production Build
1. **Request from Bryan:**
   - Hi-res logo file (vector/SVG preferred)
   - Professional headshot (hi-res)
   - Any personal photos he wants on the site (office, team, etc.)
   - Written permission to use logo and likeness

2. **Image Sourcing Strategy:**
   - Use Unsplash (free) or purchase from Shutterstock/Adobe Stock
   - Consider commissioning a professional photographer for:
     - Office/location shots in Hudson, OH
     - Updated headshot of Bryan
     - Lifestyle shots that feel authentic (not stock)
   - Budget estimate: $200-500 for stock licenses, $500-1500 for custom photography

3. **Legal Safeguards:**
   - Keep documentation of all image licenses
   - Never use FMG Suite images — even if they "look generic," the license is platform-specific
   - Unsplash images are safe for commercial use without attribution (though attribution is appreciated)
   - If using any paid stock, keep receipt/license confirmation
